# 线段树
支持解决的问题
- 区间查询，修改，维护
- 复杂度 $O(\log n)$ （平衡二叉树）
> 区间求和、最值等

不支持的问题
- 区间覆盖，区间添加
> 区间众数，区间最长连续问题、最长不下降问题等
## 基本思想
```
                   [l, r]
                  /       \
          [l, mid]         [mid+1, r]
         /      \            /       \
[l, mid1] [mid1+1, mid] [mid+1, mid2] [mid2+1, r]
```

## Lazy 标记
- 如果更新完全覆盖当前区间，更新该区间并 lazy 标记
- 否则先下传 lazy 标记，再分别搜索左右儿子，最后更新当前区间
- (即下传停止在 lazy 处)
